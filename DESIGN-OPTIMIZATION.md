## 三、光影系统设计 ✨

### 3.1 阴影层级

```css
/* 柔和阴影 - 适用于悬浮元素 */
.shadow-soft {
  box-shadow: 
    0 2px 8px rgba(92, 79, 58, 0.08),
    0 8px 24px rgba(92, 79, 58, 0.06);
}

/* 中等阴影 - 适用于卡片 */
.shadow-medium {
  box-shadow: 
    0 4px 12px rgba(92, 79, 58, 0.1),
    0 16px 48px rgba(92, 79, 58, 0.08);
}

/* 深层阴影 - 适用于浮动元素 */
.shadow-deep {
  box-shadow: 
    0 8px 24px rgba(92, 79, 58, 0.12),
    0 32px 64px rgba(92, 79, 58, 0.1);
}
```

### 3.2 光影效果

- **内发光** - 增加元素深度
- **外发光** - 强调重要元素
- **渐变边框** - 视觉引导
- **光泽滑动** - 交互反馈
- **玻璃拟态** - 现代感+层次感

## 五、交互设计增强 🎬

### 5.1 动画系统

```css
/* 悬浮动画 */
@keyframes float {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-15px); }
}

/* 脉冲发光 */
@keyframes pulse-glow {
  0%, 100% { opacity: 0.6; transform: scale(1); }
  50% { opacity: 1; transform: scale(1.05); }
}

/* 渐变流动 */
@keyframes gradient-shift {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}

/* 滑入动画 */
@keyframes slide-up {
  from { opacity: 0; transform: translateY(50px); }
  to { opacity: 1; transform: translateY(0); }
}
```

### 5.2 按钮系统

### 5.3 卡片效果

## 七、心理学设计原则 🧠

### 7.1 色彩心理学应用

| 色彩 | 情感 | 应用场景 |
|------|------|----------|
| 金色 | 成功、优质 | 统计数据、成就展示 |
| 蓝色 | 专业、信任 | 服务介绍、技术细节 |
| 绿色 | 成长、安全 | 成功案例、保障承诺 |
| 橙色 | 活力、行动 | CTA按钮、行动号召 |

### 7.2 视觉层次

1. **主次分明：** 大标题 → 小标题 → 正文 → 辅助文字
2. **色彩引导：** 渐变吸引 → 单色强调 → 中性衬托
3. **空间节奏：** 疏密有致、留白得当
4. **动线设计：** 视线自然流动、循序渐进

### 7.3 用户体验优化

- **首屏印象：** 强大的视觉冲击
- **内容消费：** 舒适的阅读体验
- **交互反馈：** 即时、明确的响应
- **情感共鸣：** 专业中不失温度
